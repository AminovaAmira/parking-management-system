### 1️⃣ OCR распознавание номеров

```python
def extract_license_plate_from_image(image_bytes):
    """Распознавание российского автомобильного номера"""
    # Загрузка изображения
    img = cv2.imdecode(np.frombuffer(image_bytes), cv2.IMREAD_COLOR)

    # Предобработка: оттенки серого
    gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)

    # Распознавание с Tesseract OCR
    text = pytesseract.image_to_string(
        gray,
        config='--psm 7',  # Single line mode
        lang='rus+eng'
    )

    # Форматирование под российский стандарт
    return format_russian_plate(text)  # Результат: А123ВС777
```

### 2️⃣ Система предоплатного баланса

**Основные операции:**
- **Пополнение** — добавление средств на баланс (карта, наличные)
- **Списание** — автоматическое при создании бронирования
- **Возврат** — при отмене до начала бронирования
- **Штраф** — за превышение времени парковки
- **Оптимистичное обновление UI** — мгновенное отображение изменений

**Схема работы:**
Бронирование → Расчет стоимости → Проверка баланса → Списание →
→ Transaction (type="booking_payment") → Обновление баланса
